# Interactive Games

Koleksi game edukatif interaktif berbasis web

## Games

| Game | Deskripsi | Status |
|---|---|---|
| 🧠 Memory Game | Latihan working memory & daya ingat visual | 🚧 In Progress |
| 👁️ Perception Game | Latihan kecepatan persepsi & ketelitian | 🚧 In Progress |
| ⚠️ Risk Quest | Uji pemahaman manajemen risiko | 🚧 In Progress |

## Tech Stack

- HTML5
- CSS3 (vanilla, no framework)
- JavaScript (vanilla, no framework)
- Hosted via GitHub Pages

## Tujuan Proyek

Proyek ini dibuat sebagai bagian dari portofolio pengembangan aplikasi web,
sekaligus media pembelajaran interaktif yang menghubungkan artikel Medium
dengan pengalaman belajar yang lebih engaging.
